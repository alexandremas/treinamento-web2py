### 1. Sobre o Python

- Linguagem de programação interpretada
- Fracamente tipada

###	2. Estruturas de dados do Python

-	Caracteres (string)

``` python
texto = 'String'	# string
```

-	Inteiro longo (long int)

``` python
valor = 3	# integer
```

-	Ponto flutuante (float)

``` python
valor = 3.14
```

-	Lista (list)

``` python
lista = [9, 'Pedone', 3.14]
```

-	Tupla (tuples)

``` python
tupla = (9, 'Pedone', 3.14)
```

-	Dicionários (dictionaries)

``` python
dicionario = {'banana':3.99, 'maca':2.99}
```

### 3. Estruturas de controles

valor = 3
